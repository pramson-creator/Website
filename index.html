Learn IT, Cybersecurity & Computer Science for Free or Very Cheap

This website is a simple roadmap for beginners who want to break into tech without going into debt. It focuses on three main tracks: IT/help desk, cybersecurity, and computer science. Each path includes explanations, starter steps, and curated free or low-cost resources.

Choose Your Tech Path

IT & Help Desk

Perfect if you like solving technical problems for people and working with devices, networks, and basic troubleshooting.

  • Help desk support
  • Desktop / laptop support
  • Entry-level IT technician

Getting Started This Week (for $0)

Step 1: Pick a Path

Visit each path page using the navigation at the top. Read the short descriptions and decide which one feels most interesting: helping people with tech (IT), defending systems (cybersecurity), or building software (computer science).

Step 2: Choose 1–2 Resources

Don’t try to do everything at once. Pick one main resource and one backup. Commit to studying a small amount every day (even 20–30 minutes can make progress over time).
